# Security Policy

## Supported Versions

This project is currently in the `0.x` development stage.

Only the latest released minor version is supported with security fixes.

| Version | Supported |
| ------- | --------- |
| 0.3.x   | ✅        |
| < 0.1   | ❌        |

> Note: The `main` branch may contain unreleased development changes for the next milestone and is not considered a supported release version.

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

If you discover a security issue, please report it privately using **GitHub Private Vulnerability Reporting**.

When reporting a vulnerability, please include:

- a clear description of the issue
- steps to reproduce it
- the affected version, branch, or commit if known
- the possible impact
- any suggested fix or mitigation, if available

## Response Process

As this is a solo-maintained project, response times may vary.

I will make a best effort to:

1. acknowledge valid reports within **7 days**
2. investigate and confirm the issue
3. prepare a fix or mitigation when appropriate
4. publish a patched release if needed
5. disclose the issue responsibly after a fix is available

## Responsible Disclosure

Please do not publicly disclose the vulnerability until the issue has been reviewed and, when possible, a fix or mitigation has been prepared.

Responsible disclosure helps protect users and keeps the project safer for everyone.

## Scope

This policy applies to the released source code in this repository.

Please note:

- this is a **solo-maintained** project
- this project is developed primarily for **educational and experimental purposes**
- unreleased code on development branches may change frequently
- third-party dependency issues may depend on upstream maintainers

## Project Context

Autonomous Paper Trader is a paper trading / backtesting project.

It is **not** intended for live trading, brokerage execution, or production financial use.
